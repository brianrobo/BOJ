
## stack

* about switch statements
  - only integer
  - break
``` c++

	while (T--) {
		cin >> val;
    
		switch (val) { //<-- only integer type..
		case 1:
			cout << "1~" << endl;
			break;
		case 2:
			cout << "2~" << endl;
			break;
		case 3:
			cout << "3~" << endl;
			break;
		default:
			cout << "default" << endl;
		}
	}
```
